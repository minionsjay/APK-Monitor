## 4. 完全离线获取代理节点的可行性

从 PCAPdroid 数据确认：
- 代理节点是**服务端在 forwarder control 会话中推送的**
- 每天变化（不是硬编码）
- 不在 .dat 文件中（.dat 只有 3 个控制面节点）
- 要离线获取，必须模拟 forwarder control 协议
